My Flex App

MyFlix is a full-stack web application that allows users to browse movies, manage their favorites list, and update profiles—all within a secure and intuitive platform. The project features a RESTful API built with Express.js, Node.js, and MongoDB, alongside a React front-end client that ensures a seamless user experience. Deployed on Heroku, MyFlix showcases my ability to create robust, user-centered applications with secure authentication and data management.
